01 Four-Layer Architecture

The foundational structure of every AAO system. The AI can see everything but can only touch what it is explicitly permitted to touch. Each layer has a precise scope — crossing boundaries requires passing through the layer above.

03 Session Lifecycle

An AAO session begins when an operator initiates AI-assisted work and ends when the operator signs off the Session Summary. Nothing is considered complete — and no session is formally closed — until a named individual with an Authorization Code has reviewed and accepted the session's output.

04 Human Sign-Off Workflow

The sign-off workflow is the human accountability mechanism at the heart of AAO. Every consequential AI output must be reviewed and signed by a named individual using their Authorization Code before it becomes an organizational record. This is where process enforces trust.

05 Snapshot & Rollback

Before every consequential action, AAO captures a point-in-time snapshot of runtime state. If the post-action health check fails, rollback is automatic — no human intervention required. The system always returns to a known good state.

SNAPSHOT / ROLLBACK — State management and recovery

06 Prompt Injection Defence

Prompt injection is the primary attack vector for AI systems with real-world action capability. AAO's defence is structural — even a perfectly crafted injection cannot request an action that doesn't exist on the whitelist, because the whitelist is on the immutable base.

07 Deployment Authorization Process

Before an AAO system operates in production, organizational authority must sign a Deployment Authorization Document. This is a one-time act of accountability — a named executive accepts responsibility for the system and authorizes it to run. Everything after this is governed by the process.

09 Failure Intelligence Loop

AAO systems learn from production failures. Every failure is captured, analyzed for patterns, and fed back into the development cycle automatically. The difference between monitoring and intelligence is the feedback loop.
